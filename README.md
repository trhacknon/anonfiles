# ANONFILE
A PYTHON SCRIPT FOR ANONFILE UPLOADING
<h3 align="left">INSTALL :</h3>

````
pkg up -y
pkg i git -y
pkg i python -y
pip3 install lolcat
git clone https://github.com/trhacknon/anonfile
cd anonfile
python anon.py
````
AFTER THAT GIVE FILE PATH FOR EXAMPLE:
````
/sdcard/main.py
````
